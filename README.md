# CRUD Filmes

Aplicação fullstack focada em fazer um gerenciamento básico de filmes.

## Funcionalidades

- Listar filmes;
- Realizar cadastro de filme;
- Visualizar informações de um filme;
- Atualizar informações do filme;
- Apagar filme.

## Tecnologias

- Vue3;
- Vuetify;
- VueRouter;
- Axios;
- Spring Boot;
- PostgreSQL.
